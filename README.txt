ALIEN - This object is drawable and movable. It teleports randomly within a range of 5 squares. If you interact with this piece it will kill you. 

GOBLIN - This object is drawable and movable. It moves back and forth left and right and if you interact with it it hits your for one point. 

TREASURE CHEST - This object is drawable and not movable. It advances you to the next level when you interact with it.

POWER UP - This object is drawable and not movable. It gives you a point when you interact with it.

CHAIR - This object is drawable and not movable. It does nothing when you interact with it
